# AngularFHIR
FHIR/DICOMWeb demo written in AngularJS working from SIIM 2015 Hackathon codebase by Chris Hafey. This was mainly used as a learning-exercise for me to get familiar with building SPAs using Angular.

##Current Views
-Patient search  
-Patient report view  
-Study Detail view  
-Report query view
